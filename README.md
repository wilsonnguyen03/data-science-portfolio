## ⭐ Featured Projects

### 🔍 TruthLens – Forensic Media Verification Dashboard
Interactive, user-centric dashboard aggregating multimodal detection signals (image, video, audio, and text) into a unified analytical interface for media manipulation detection. Integrates state-of-the-art detection models with explainable AI (XAI) mechanisms, provenance indicators, metadata analysis, and confidence scoring — designed for non-expert users.

**Stack:** React 19 · Vite · TypeScript · Chart.js · Leaflet · Tailwind CSS · Python · APIs

👉 [View Repository](https://github.com/wilsonnguyen03/truth-lens-62)

**Outcome:** MVP UI complete with multimodal dashboards (text, image, video), propagation map, and detection pipeline stubs ready for model integration. Built for DATA7901 capstone with a focus on usability, trust calibration, and responsible AI.
